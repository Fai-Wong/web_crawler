# 爬虫集合

- crawler_01：[糗事百科](https://www.qiushibaike.com/)爬虫
- crawler_02：[淘宝mm](https://mm.taobao.com/json/request_top_list.htm)爬虫，下载每个mm的个人信息，爱秀照片，分别保存各自文件夹
